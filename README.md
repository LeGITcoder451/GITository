# 🛠️ GITository: The Interactive Git Hub

Welcome to **GITository**! This repository serves as an interactive web dashboard and a quick learning resource for mastering version control with Git.

## 🚀 Project Features
* **Live Search:** Filter common Git commands on the fly using the search input.
* **Responsive Layout:** Beautiful grid design built with CSS Grid that scales seamlessly across screens.
* **Vanilla Stack:** Written cleanly using HTML5, CSS3, and JavaScript without bloated frameworks.

## 📋 Essential Git Commands Covered
* `git init` — Initialize a local repository.
* `git status` — Check tracking changes.
* `git add .` — Stage your changes.
* `git commit -m "message"` — Commit changes locally.
* `git push origin main` — Upload changes to GitHub.
